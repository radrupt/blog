#ajax, XmlHttpRequest

异步请求，可以实现页面不整体刷新的情况下，更新数据，提升页面加载速度

readyState: 有5种状态

  1. 请求未初始化
  2. 服务器连接已建立
  3. 请求已接收
  4. 请求处理中
  5. 请求已完成，且响应已就绪
