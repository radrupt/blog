2019-02-10

字符串简称

  1. \s空白符，\S非空白符
  2. \w字母数字下划线，\W非数字字母下划线
  3. \b字母边界，\B非字母边界
  4. \d数字，\D非数字

量词

  1. {m,n}
  2. {n}
  3. +
  4. *
  5. ? 0和1

贪婪，懒惰

常规的量词都是贪婪方法，而量词后面再加上?，则是懒惰，贪婪就是尽可能多的匹配，懒惰就是尽可能少的判断

捕获,做括号优先匹配

“1Qz2Wx”.replace(/((\d)[A-Z])([a-z])/g,”$1$2$3”)//“1Q1z2W2x”

正向预查找

/q(?=p)/.test(“qp”)//true

分支,这个也是懒惰匹配

/(q|a|b)/
